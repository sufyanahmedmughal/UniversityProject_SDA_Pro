# ADR-004: Threat Intel Caching Strategy

## Status: Accepted

## Decision
Redis-backed CachingProxy with 1-hour TTL

## Reasoning
- VirusTotal free tier: 4 requests/minute
- Redis provides fast in-memory lookup (< 1ms vs 200-500ms API call)
- CachingProxy pattern keeps caching logic separate from business logic
- TTL of 1 hour balances freshness with rate limit protection
